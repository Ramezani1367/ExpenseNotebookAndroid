■ چیزهایی که می‌تونی امتحان کنی
  • ویجت ۴×۲ روی گوشی: روز/تاریخ بالا، «خرج کرد امروز» + عدد سبز، ردیف پایین آیتم‌ها
  • ⚙ چرخ‌دنده (روی هر ۳ ویجت): امروز / این هفته (از شنبهٔ همین هفته) / این ماه (از ۱ام ماه)
    ← لیبل، جمع، خط «روز | تاریخ کامل» خاکستری + بازهٔ سبز، شمارش — همه سینک
  • ردیف آیتم‌ها: ۱ تا ۳ مورد، بسته به طول عنوان/مبلغ هرچقدر جا بشن + شمارندهٔ ⁺N
  • ➕ سبز: پاپ‌آپ ثبت سریع بالای صفحه جلوی ویجت (عنوان + مبلغ با جداکنندهٔ فارسی)
  • کلیک بدنهٔ هر ویجت: اپ روی همون دورهٔ فعال باز می‌شه (هفته/ماه = گروه‌بندی روزانه)
  • تو مینی‌اپ: ضربه روی آیتم ← مدال ویرایش • 🗑 حذف • Esc زنجیره‌ای برمی‌گردونه
  • دکمه‌های سایز ۳×۲/۴×۲: شبیه‌سازی ریسایز ویجت اندروید
  • ویجت‌های ۴×۱ باریک و ۲×۲ مربع (ستون کناری): زنده و هم‌طرح، ⚙ خودشون رو دارن

■ جزئیات فنی
  • تاریخ شمسی واقعی (تابع تبدیل جلالی/میلادی) — محاسبهٔ هفته از شنبه، ماه از ۱ام
  • داده‌های دمو روی localStorage مرورگر خودت ذخیره می‌شه (دکمهٔ «↺ بازگردانی» ریست می‌کنه)
  • این فایل با ۱۲۴ تست خودکار (jsdom) سبزه ✅
  • نگاشت به نسخهٔ واقعی: Glance + RemoteViews، رنگ #1a5e3a، WorkManager برای رفرش
